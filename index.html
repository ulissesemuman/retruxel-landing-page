System Interface

The Builder

A visual IDE designed for the technical purist. Retruxel lets you place and configure game modules on a scene canvas, wire them to hardware events, and compile directly to authentic ROM files — no C required.

  • Integrated Sprite & Map Editor
  • Event-Based Module System
  • One-Click ROM Build & Export

Zero Setup Architecture

Retruxel ships with SDCC and cc65 toolchains for all supported targets. They are extracted automatically on first build — no manual compiler setup, no environment variables.

Module System

Drag and drop pre-built modules — physics, input, entities, animation — onto the scene canvas and wire them to hardware events like VBlank and OnStart.

One-Click ROM Export

Produce hardware-accurate ROM files instantly. Ready for flashcarts or professional distribution.

Ready for Real Hardware

Retruxel outputs valid ROM files that run on original consoles and flashcarts. The generated C code targets real hardware constraints — VDP limits, scanline budgets, and Z80 or 6502 memory maps.

Architecture

Configure Once, Build Anywhere

Every target in Retruxel ships with its own build profile — ROM size, region timing, emulator path, and launch behavior. Set it once per platform and Retruxel handles the rest at build time.

Modules Compile Per Target

Each module has a dedicated code generator per platform. The same Physics or Input module produces correct C code for SMS, NES, or Game Gear automatically.

Third-Party Plugin Support

The Retruxel SDK allows developers to ship custom modules and targets as plugins, extending the tool without modifying the core.
